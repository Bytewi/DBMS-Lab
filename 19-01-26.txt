DDL:
1)  Create : To create tables , procedures , triggers etc.
2)  Alter : Modify the structure of table.
3)  Truncate : To delete all the data but not the Table.
4)  Drop : Table will be deleted.
5)  Rename : Rename tables , columns in DBMS.

DML:
1)  Insert Command : Data into the table.
2)  Update Command : Update the columns.
